# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in fabprint, please report it
responsibly by emailing **paul@fremantle.org**.

Please do **not** open a public GitHub issue for security vulnerabilities.

## Scope

fabprint interacts with external services (Bambu Lab Cloud API, local printers
via LAN/MQTT, Moonraker API). Security reports related to credential handling,
network communication, or command injection are especially welcome.
